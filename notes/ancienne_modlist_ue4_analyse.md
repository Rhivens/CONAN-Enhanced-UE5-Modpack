# Ancienne modlist UE4 - Analyse

## Objectif

Conserver une trace de l'analyse de l'ancienne modlist Conan Exiles UE4 sans la reprendre directement sur Enhanced / UE5.

## Conclusion générale

L'ancienne modlist UE4 avait une bonne philosophie :

- qualité de vie ;
- interface ;
- construction ;
- navigation ;
- cosmétiques ;
- confort de jeu ;
- enrichissement de l'expérience sans chercher à transformer complètement Conan.

## Décision

La modlist ne doit pas être reprise telle quelle.

Elle servira uniquement de référence pour identifier :

- les besoins à conserver ;
- les types de mods intéressants ;
- les catégories prioritaires ;
- les idées à rechercher en version compatible Enhanced / UE5.

## À compléter

Ajouter progressivement :

- anciens mods utiles à retrouver ;
- anciens mods à abandonner ;
- équivalents UE5 éventuels ;
- notes de compatibilité.
