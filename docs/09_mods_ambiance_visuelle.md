## Objectif

Documenter les mods liés à l'ambiance visuelle, à la météo, à la lumière et aux éventuels réglages graphiques complémentaires.

## Décision actuelle

ReShade n'est pas réinstallé pour le moment, car la base UE5 est déjà jugée visuellement satisfaisante.

## Critères de sélection

Les mods ou réglages de cette catégorie devront :

- être compatibles Conan Exiles Enhanced / UE5 ;
- ne pas détériorer les performances ;
- éviter les conflits avec Lumen et les nouveaux systèmes UE5 ;
- apporter une amélioration réelle plutôt qu'un simple effet cosmétique lourd ;
- être testés avec les réglages actuels : Ultra, DLSS Qualité, Frame Generation, Reflex, gamma 1.8.
