# 13 - Mods NSFW

## Objectif

Documenter les mods NSFW intégrés au modpack Conan Exiles Enhanced / UE5.

Ce module est destiné à une partie privée entre adultes.

## Règle générale

Les mods NSFW doivent être ajoutés avec prudence :

- un mod à la fois ou par très petit lot ;
- test en host local solo avant validation ;
- documentation claire de la méthode d'utilisation ;
- vigilance particulière sur les interactions avec les personnages, thralls, animations et mécaniques multijoueur ;
- information claire des joueurs avant installation.

La nudité totale est activée par défaut dans le cadre du modpack.

## Mods validés

### Primal Instinct

- Catégorie : NSFW / contenu adulte.
- Source : Steam Workshop.
- Workshop ID : `3723050568`.
- Lien : https://steamcommunity.com/sharedfiles/filedetails/?id=3723050568

#### Résultat du test

- Abonnement Steam Workshop effectué.
- Mod activé depuis le menu Mods en jeu.
- Test réalisé en host local solo.
- Fonctionnement validé ingame après prise en main du mod.

#### Note d'utilisation

Le mod ne se présente pas comme un simple menu de réglages classique.

Il demande une prise en main spécifique en jeu afin de comprendre son fonctionnement et ses options.
