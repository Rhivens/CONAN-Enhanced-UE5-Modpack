# 90 - Décisions différées

## Objectif

Centraliser les décisions reportées afin de ne pas bloquer la progression du projet tout en gardant une trace claire des points à reprendre.

## ReShade

Statut : **différé**.

Décision actuelle :

- ne pas réinstaller ReShade immédiatement ;
- conserver la base visuelle UE5 actuelle ;
- réévaluer plus tard uniquement si un besoin réel apparaît.

Raison :

- la base UE5 est déjà jugée propre et fluide ;
- Lumen, DLSS Qualité, Frame Generation et Reflex donnent déjà un bon résultat ;
- éviter d'ajouter une couche externe tant que la stabilité du modpack n'est pas construite.

## Ancien fichier Better Graphics UE4

Statut : **non repris tel quel**.

Décision actuelle :

- ne pas réutiliser directement l'ancien fichier ;
- s'en servir éventuellement comme référence historique ;
- reconstruire les réglages spécifiquement pour Enhanced / UE5 si nécessaire.

## Test multijoueur réel Conan UE5

Statut : **différé, non bloquant**.

Raison :

- aucun ami disponible actuellement pour le test ;
- host local solo déjà validé ;
- expérience solide d'hébergement multi/coop ;
- connexion Free 8 Gb/s stable ;
- objectif Conan limité à 3 ou 4 joueurs.

À faire plus tard :

- test avec 1 ami ;
- test après premiers mods ;
- test avant validation finale du modpack.
