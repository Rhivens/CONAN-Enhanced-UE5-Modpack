# 10 - Mods Administration & Host local

## Objectif

Documenter les mods et réglages utiles à l'administration d'une partie privée en host local.

## Statut

**À démarrer.**

## Contexte

Le projet vise une partie privée hébergée localement sur le PC principal, pour environ 3 à 4 joueurs.

## Critères de sélection

Les mods ou réglages de cette catégorie devront :

- faciliter l'administration sans alourdir inutilement la partie ;
- rester adaptés à un host local privé ;
- éviter les outils pensés uniquement pour de gros serveurs publics ;
- ne pas compliquer les sauvegardes ou la stabilité ;
- être testés avant validation.
