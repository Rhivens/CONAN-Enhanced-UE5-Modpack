# 02 - Tests graphiques et performances

## Objectif

Valider que la base Conan Exiles Enhanced / UE5 est visuellement satisfaisante et techniquement fluide avant l'ajout de mods.

## Réglages testés

- Résolution : 1920x1080.
- Preset : Ultra.
- DLSS : Qualité.
- Génération d'images DLSS : activée.
- Nvidia Reflex : activé.
- Lumen : Ultra.
- Ombres : Ultra.
- Textures : Ultra.
- Feuillages : Ultra.
- Effets : Ultra.
- Gamma : ajusté à 1.8.

## Résultat visuel

- Image jugée propre.
- Netteté correcte après ajustement.
- Ombres satisfaisantes.
- Végétation au loin correcte.
- Caméra ajustée selon le confort de jeu.

## Résultat performances

- FPS volontairement capés à 90.
- Aucun ralentissement notable.
- Aucun stutter constaté.
- Températures normales.

## Décision

ReShade n'est pas nécessaire pour le moment.

## Statut

**Validé.**
