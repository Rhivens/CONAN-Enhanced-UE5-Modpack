# 08 - Mods Cosmétiques, Armures & Vêtements

## Objectif

Documenter les mods liés à l'apparence des personnages, aux armures, aux vêtements et aux éléments cosmétiques.

## Statut

**En cours.**

Mods actuellement validés :

1. **Fashionist v5.0.2-enhanced**.
2. **Topfreedom Enhanced**.
3. **Tanny's WDV Tattoos und Narben - Enhanced**.
4. **Sacred Lust armor set**.

## Critères de sélection

Les mods de cette catégorie devront :

- être compatibles Conan Exiles Enhanced / UE5 ;
- ne pas casser les apparences, animations ou équipements ;
- éviter les conflits entre packs d'armures ;
- rester cohérents avec l'ambiance souhaitée ;
- être testés en jeu avant validation ;
- rester compatibles avec l'objectif coop / host local privé.

## Mods candidats

À compléter.
