# 08 - Mods Cosmétiques, Armures & Vêtements

## Objectif

Documenter les mods liés à l'apparence des personnages, aux armures, aux vêtements et aux éléments cosmétiques.

## Statut

**En cours.**

Premier mod cosmétique validé : **Fashionist v5.0.2-enhanced**.

## Critères de sélection

Les mods de cette catégorie devront :

- être compatibles Conan Exiles Enhanced / UE5 ;
- ne pas casser les apparences, animations ou équipements ;
- éviter les conflits entre packs d'armures ;
- rester cohérents avec l'ambiance souhaitée ;
- être testés en jeu avant validation.

## Mods validés

### Fashionist v5.0.2-enhanced

- Catégorie : Cosmétique / armures / apparence.
- Source : Steam Workshop.
- Workshop ID : `3720921242`.
- Type : gestion de l'apparence, des armures, des armes et options visuelles du personnage.
- Statut : **validé en jeu**.

#### Résultat du test

- Abonnement Steam Workshop effectué.
- Mod activé depuis le menu Mods en jeu.
- Test réalisé en host local solo dans un pack avec Better Thralls.
- Fonctionnement validé ingame.

#### Conclusion

**Fashionist est validé comme premier mod du bloc Cosmétiques, Armures & Vêtements.**
